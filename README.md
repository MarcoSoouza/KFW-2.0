# KFW LTDA Website

## Project Description
This is the official website for KFW LTDA, a company specializing in mining equipment rental and maintenance. The website showcases the company's services, clients, and contact information.

## Company Information
- **Name:** KFW LTDA
- **Industry:** Mining Equipment Rental and Maintenance
- **Services:**
  - Equipment Rental
  - Mechanical Maintenance
  - Electrical Maintenance
  - Machine Refurbishment
- **Experience:** Over 15 years serving major mining companies
- **Standards:** International quality standards

## Project Structure
- `index.html`: Main HTML file with website content
- `style.css`: Stylesheet for design and layout
- `script.js`: JavaScript for carousel functionality
- `logos/`: Folder for client logo images
- `TODO.md`: Task list for project organization
- `README.md`: This file

## Design Features
- Responsive design for mobile and desktop
- Carousel for client logos
- Clean, professional layout using Poppins font
- Color scheme: Yellow (#F7B928), Black (#1A1A1A), Gray (#7D7D7D)

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## Setup
1. Ensure all files are in the same directory
2. Place client logo images in the `logos/` folder
3. Open `index.html` in a web browser
